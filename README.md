# Didi Korean Podcast Landing Page

This project is a rebuilt landing page for Didi Korean Podcast. It keeps the original site destinations and embedded YouTube video while reworking the brand image into a fuller page design with green backgrounds, handwritten typography, tape details, and a portrait-focused visual system.

## Tech Stack

- TanStack Start
- React 19
- TanStack Router
- Tailwind CSS 4 with custom global CSS
- Lucide React icons
- Netlify

## Local Development

Install dependencies and start the local dev server:

```bash
npm install
npm run dev
```

The Netlify deployment uses the build configuration in `netlify.toml`.
